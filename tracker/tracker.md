# Progress Tracker

## How to Use

Score each area weekly (Sunday, during your weekly review). Use a simple 1-5 scale.

| Score | Meaning |
|-------|---------|
| 1 | Didn't practice at all |
| 2 | 1-2 days |
| 3 | 3-4 days |
| 4 | 5-6 days |
| 5 | Every day |

## Rollout Plan (Don't Try Everything At Once)

**Week 1-2:** Foundations only

- Sleep protocol
- Exercise
- Morning no-phone
- Evening gratitude

**Week 3-4:** Add Layer 2

- Meditation
- CBT journaling (one entry/day)
- Intention setting

**Week 5-6:** Add Layer 3

- Deep work blocks with reward protocol
- Environmental design changes
- Implementation intentions

**Week 7+:** Full system

- All practices active
- Weekly review
- Social environment strategy

**The rule:** Don't add the next tier until the current tier scores 3+ on most items.

## Tool Recommendations

This tracker works as markdown. For better daily tracking and sharing:

**Best fit: Notion**

- Import these markdown files directly (Notion supports md import)
- Create a daily database with the checklist as properties
- Dashboard view for weekly/monthly trends
- Shareable via link with colleagues
- Templates for repeating the daily checklist

**Alternatives:**

- **Obsidian** — if you prefer local-first, private, and markdown-native

- **Apple Shortcuts + Health** — for automating sleep/exercise tracking

- **Streaks app (iOS)** — simple habit streak tracking with visual feedback

- **Loop Habit Tracker (Android)** — open-source, graphs, no account needed

**For the CBT journal specifically:**

- Paper works surprisingly well (slower = more processing time = better reappraisal)
- Or a dedicated section in Notion with the 4-column template as a database

**For feedback on how things are going:**

- Your weekly review IS the feedback mechanism
- The tracker totals over time show trajectory
- Monthly reflection questions surface qualitative insights
- If you want external feedback: share your tracker with a trusted friend and review together monthly (combines social accountability + the "trusted advisor" practice)
